## Compilare con bash
1. andare nella giusta direcotory
2. gcc *nome_file* -o *nome_file*.out
3. ./*nome_file*.out

NB: ad ogni modifica bisogna re-compilare il file prima di runnarlo (in più il tasto run non serve più)
